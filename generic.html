COVID-19 TRACKER
PROJECT

A simple, yet effective web app that tracks the cases, recoveries and deaths of coronavirus around the world. Built using ReactJS..

FUNCTIONS:

  • Worldwide count and statistics of cases.
  • Specific count of country through the navbar list.
  • Map that zooms in on chosen country.
  • All built using ReactJS and deployed on Firebase.
